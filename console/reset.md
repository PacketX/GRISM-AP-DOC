# reset

## Commands

- `reset all`: Reset all settings to defaults
- `reset allowList`: Reset the firewall allow list (Allow IP List) to defaults
- `reset account`: Reset account settings to defaults

## Tip

You can also use `reset help` / `reset ?` to list available keys.

## Example: `reset help` Output

```text
> reset help
Usage: reset <key>
Available keys:
  all         - Reset all settings to default
  allowList   - Reset firewall allow list to default
  account     - Reset user accounts to default
```
