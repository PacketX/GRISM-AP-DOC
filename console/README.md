# Console Commands

This section lists GRISM AP console commands, grouped by purpose:

- Global: common system and status commands
- `set`: configuration commands (management interface and service toggles)
- `reset`: restore settings to defaults

> Actual supported commands may vary by device/firmware. Use `help` / `?`, `set help`, or `reset help` to see available keys.
